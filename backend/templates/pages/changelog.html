Changelog

A running log of notable changes to the OpenBlog platform.

v0.3 — February 2025

  • Added global search with Posts and Topics tabs and live suggestions.
  • Added password reset via email with 1-hour expiry tokens.
  • Email verification now gates posting and commenting.
  • NIST-compliant 15-character minimum password requirement.
  • Show/hide toggle on all password fields.
  • Updated footer with navigation links and legal pages.

v0.2 — January 2025

  • Full design system overhaul: Inter UI font, Source Serif 4 reading font.
  • Prose typography pass: 18px reading size, improved heading scale.
  • Diff/revision viewer redesign with unified and split modes.
  • Added internationalization (English and Spanish).
  • Docker Nginx configuration fixes for reliable production deployments.

v0.1 — December 2024

  • Initial release: posts, revisions, comments, voting, badges.
  • JWT API + session-based SSR authentication.
  • Celery + Redis task queue for scheduled publishing and analytics.
  • Prometheus metrics endpoint.
